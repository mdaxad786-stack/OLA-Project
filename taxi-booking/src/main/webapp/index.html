Steps to Install and configure Ansible 1. Create an user called ansadmin useradd ansadmin 2. Setup the passoword for ansadmin passwd ansadmin -- enter passoword 3. Add ansadmin user to sudoers file visudo ansadmin ALL=(ALL) NOPASSWD: ALL 4. Enable passoword based authentication in sshd_config file vi /etc/ssh/sshd_config PasswordAuthentication yes 5. Reload sshd_config file service sshd reload 6. Change to ansadmin user su - ansadmin 7. Create ssh keys ssh-keygen 8. Install Ansible yum install ansible Default path is cd /etc/ansible 9. Repeat step 1 to 6 in all the target nodes 10.Copy the public key to Target Nodes ssh-copy-id ansadmin@PrivateIP 10. Check if your ansible node is able to communicate with the target nodes ansible all -m pin
